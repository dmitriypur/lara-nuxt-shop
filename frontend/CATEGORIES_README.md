# Страницы категорий

## Описание
Добавлена функциональность для работы с категориями товаров в интернет-магазине.

## Новые страницы

### 1. `/categories` - Список всех категорий
- Отображает все доступные категории товаров
- Поиск по названию и описанию категорий
- Переход к конкретной категории по клику
- Адаптивный дизайн с карточками категорий

### 2. `/categories/[slug]` - Страница конкретной категории
- Отображает товары выбранной категории
- Фильтрация и поиск товаров в рамках категории
- Сортировка по цене, названию, рейтингу
- Фильтр по цене и наличию
- Переключение между сеткой и списком товаров
- Пагинация результатов

## Компоненты

### Breadcrumbs.vue
- Компонент хлебных крошек для навигации
- Автоматически генерирует путь навигации
- Поддерживает ссылки и текущую страницу

## Обновленные компоненты

### AppHeader.vue
- Добавлена ссылка "Категории" в основную навигацию
- Добавлена ссылка в мобильное меню

### ProductCard.vue
- Добавлена ссылка на категорию товара
- Отображается как бейдж под названием товара

### ProductListItem.vue
- Добавлена ссылка на категорию товара
- Отображается как бейдж под названием товара

### AppFooter.vue
- Обновлен раздел "Каталог" с ссылками на категории

## API интеграция

Используются существующие методы API:
- `getCategories()` - получение списка категорий
- `getCategory(slug)` - получение конкретной категории
- `getProductsByCategory(slug, params)` - получение товаров по категории

## Маршрутизация

- `/categories` → `pages/categories/index.vue`
- `/categories/[slug]` → `pages/categories/[slug].vue`
- `/products` → `pages/products/index.vue`
- `/products/[slug]` → `pages/products/[slug].vue`

## Особенности реализации

1. **SEO оптимизация**: Динамические заголовки и мета-теги для каждой категории
2. **Кэширование**: Использование `useLazyAsyncData` для оптимизации загрузки
3. **URL параметры**: Сохранение состояния фильтров в URL для возможности поделиться ссылкой
4. **Адаптивность**: Полностью адаптивный дизайн для всех устройств
5. **Навигация**: Хлебные крошки на всех страницах для удобной навигации

## Использование

1. Перейти на `/categories` для просмотра всех категорий
2. Кликнуть на категорию для просмотра товаров в ней
3. Использовать фильтры и поиск для навигации по товарам
4. Переходить между категориями через навигацию или хлебные крошки

## Стили

Все стили используют существующую систему CSS классов:
- `btn-primary`, `btn-secondary` для кнопок
- `card` для карточек
- `input-field` для полей ввода
- Адаптивная сетка с Tailwind CSS
