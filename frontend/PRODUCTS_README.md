# Страницы товаров

## Описание
Добавлена функциональность для работы с товарами интернет-магазина.

## Новые страницы

### 1. `/products` - Список всех товаров
- Отображает все доступные товары
- Поиск по названию товаров
- Фильтрация по категориям
- Сортировка по цене, названию, рейтингу
- Фильтр по цене и наличию
- Переключение между сеткой и списком товаров
- Пагинация результатов

### 2. `/products/[slug]` - Страница конкретного товара
- Детальная информация о товаре
- Галерея изображений с миниатюрами
- Информация о категории
- Варианты товара (если есть)
- Выбор количества
- Добавление в корзину
- Добавление в избранное
- Характеристики товара
- Подробное описание
- Похожие товары

## Компоненты

### ProductCard.vue
- Карточка товара для сеточного отображения
- Ссылка на страницу товара по slug
- Ссылка на категорию товара
- Кнопка добавления в корзину
- Кнопка добавления в избранное

### ProductListItem.vue
- Строка товара для спискового отображения
- Ссылка на страницу товара по slug
- Ссылка на категорию товара
- Расширенная информация о товаре
- Кнопки действий

## Обновленные компоненты

### AppHeader.vue
- Добавлена ссылка "Товары" в основную навигацию
- Добавлена ссылка в мобильное меню

### AppFooter.vue
- Обновлен раздел "Каталог" с ссылками на товары

### Breadcrumbs.vue
- Хлебные крошки для навигации по товарам

## API интеграция

Используются методы API:
- `getProducts(params)` - получение списка товаров с фильтрами
- `getProduct(slug)` - получение конкретного товара по slug
- `getCategories()` - получение категорий для фильтров

## Маршрутизация

- `/products` → `pages/products/index.vue`
- `/products/[slug]` → `pages/products/[slug].vue`

## Особенности реализации

1. **SEO оптимизация**: Динамические заголовки и мета-теги для каждого товара
2. **Кэширование**: Использование `useLazyAsyncData` для оптимизации загрузки
3. **URL параметры**: Сохранение состояния фильтров в URL для возможности поделиться ссылкой
4. **Адаптивность**: Полностью адаптивный дизайн для всех устройств
5. **Навигация**: Хлебные крошки на всех страницах для удобной навигации
6. **Корзина**: Интеграция с системой корзины
7. **Избранное**: Сохранение состояния в localStorage

## Использование

1. Перейти на `/products` для просмотра всех товаров
2. Использовать фильтры и поиск для навигации
3. Кликнуть на товар для просмотра деталей
4. На странице товара выбрать количество и добавить в корзину
5. Переходить между товарами через навигацию или хлебные крошки

## Стили

Все стили используют существующую систему CSS классов:
- `btn-primary`, `btn-secondary` для кнопок
- `card` для карточек
- `input-field` для полей ввода
- Адаптивная сетка с Tailwind CSS
- Анимации загрузки и переходов
